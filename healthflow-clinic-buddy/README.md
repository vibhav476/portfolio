# HealthFlow Clinic Buddy

An AI-powered clinic management assistant that streamlines healthcare workflows, patient management, and clinic operations.

## Live Demo

[healthflow-clinic-buddy.lovable.app](https://healthflow-clinic-buddy.lovable.app)

## Overview

HealthFlow Clinic Buddy is designed to help healthcare professionals manage their day-to-day clinic operations more efficiently. It leverages AI to assist with patient flow, appointment management, and clinical workflow optimization.

## Key Features

- AI-assisted patient management
- Clinic workflow optimization
- Clean, intuitive healthcare UI
- Built for speed and simplicity

## Tech Stack

- **Frontend:** React (built with Lovable AI)
- **Hosting:** Lovable Cloud
